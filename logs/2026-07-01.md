## 🧠 Challenge of the Day

### Count Vowels

**Difficulty:** Easy
**Topic:** Strings / Regex

**Problem:**

Count the number of vowels (a, e, i, o, u) in a string. Case-insensitive.

**Example:**

```
Input: "Hello World"
Output: 3  (e, o, o)
```

**Hint:**

Match against a vowel regex, or iterate and check membership in a Set.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
// Regex approach
function countVowels(str) {
  return (str.match(/[aeiou]/gi) || []).length;
}

// Set approach (faster for very long strings)
function countVowelsSet(str) {
  const vowels = new Set(['a','e','i','o','u']);
  let count = 0;
  for (const ch of str.toLowerCase()) {
    if (vowels.has(ch)) count++;
  }
  return count;
}
```

**Explanation:** Two valid approaches. Regex is concise; Set is faster because Set.has() is O(1) and there's no regex compilation.

## 💡 Tip of the Day

**Top-level `await` works in ES modules**

In `.mjs` files (or with `"type": "module"` in package.json), you can use `await` at the top level — no wrapping IIFE needed:

```js
// top-level await in app.mjs
const data = await fetch('/api/data').then(r => r.json());
console.log(data);
```

Useful for scripts and config files. Doesn't work in CommonJS (`.cjs` / plain `.js`).
