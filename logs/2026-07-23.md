## 🧠 Challenge of the Day

### Valid Parentheses

**Difficulty:** Easy
**Topic:** Stacks

**Problem:**

Given a string containing just the characters `(`, `)`, `{`, `}`, `[`, `]`, determine if the input string is valid. Open brackets must be closed by the same type, in the correct order.

**Example:**

```
Input: "()[]{}"
Output: true

Input: "([)]"
Output: false
```

**Hint:**

Use a stack. Push opening brackets, pop when you see a closing bracket and check it matches.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function isValid(s) {
  const stack = [];
  const pairs = { ')': '(', '}': '{', ']': '[' };
  for (const ch of s) {
    if (ch === '(' || ch === '{' || ch === '[') {
      stack.push(ch);
    } else {
      if (stack.pop() !== pairs[ch]) return false;
    }
  }
  return stack.length === 0;
}
```

**Explanation:** Stacks are perfect for nested/matching problems. Last-opened bracket must be first-closed — LIFO.

## 💡 Tip of the Day

**Use `Number.isFinite()` not `isFinite()`**

Global `isFinite()` coerces — `isFinite('42')` returns `true`. `Number.isFinite()` doesn't.

```js
isFinite('42');          // true (coerces string to number)
Number.isFinite('42');   // false (strict type check)
Number.isFinite(42);     // true
Number.isFinite(Infinity); // false
```
