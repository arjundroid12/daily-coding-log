## 🧠 Challenge of the Day

### Factorial

**Difficulty:** Easy
**Topic:** Recursion

**Problem:**

Compute n! (n factorial) — the product of all positive integers up to n. Note: 0! = 1.

**Example:**

```
Input: 5
Output: 120  (5 × 4 × 3 × 2 × 1)
```

**Hint:**

Either recursively (n × factorial(n-1), base case 0! = 1) or iteratively with a running product.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
// Recursive
function factorial(n) {
  if (n <= 1) return 1;
  return n * factorial(n - 1);
}

// Iterative (no stack overflow risk)
function factorialIter(n) {
  let result = 1;
  for (let i = 2; i <= n; i++) result *= i;
  return result;
}
```

**Explanation:** Recursion is elegant but for large n, iterative is safer (JavaScript has a call stack limit around 10,000).

</details>

## 💡 Tip of the Day

**Use `Intl` APIs for locale-aware formatting**

Stop hardcoding date/number formats. Use `Intl`:

```js
// Currency
new Intl.NumberFormat('en-IN', { style: 'currency', currency: 'INR' })
  .format(1234567.89);  // '₹12,34,567.89' (Indian grouping)

// Dates
new Intl.DateTimeFormat('en-IN', { dateStyle: 'full' })
  .format(new Date());  // 'Monday, 29 June 2026'

// Relative time
new Intl.RelativeTimeFormat('en', { style: 'short' })
  .format(-1, 'day');  // 'yesterday'
```
