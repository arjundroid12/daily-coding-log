## 🧠 Challenge of the Day

### Palindrome Check

**Difficulty:** Easy
**Topic:** Strings

**Problem:**

Check if a string is a palindrome, ignoring case, spaces, and non-alphanumeric characters.

**Example:**

```
Input: "A man, a plan, a canal: Panama"
Output: true
```

**Hint:**

Sanitize the string first (lowercase, strip non-alphanumeric), then use two pointers from both ends.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function isPalindrome(str) {
  const clean = str.toLowerCase().replace(/[^a-z0-9]/g, '');
  let left = 0, right = clean.length - 1;
  while (left < right) {
    if (clean[left] !== clean[right]) return false;
    left++;
    right--;
  }
  return true;
}
```

**Explanation:** Cleaning the string first makes the comparison straightforward. Two-pointer avoids creating a reversed copy.

## 💡 Tip of the Day

**`Promise.allSettled()` waits for all, never rejects**

Unlike `Promise.all()` which rejects on the first failure, `Promise.allSettled()` waits for all and reports status:

```js
const results = await Promise.allSettled([
  fetch('/api/a'),
  fetch('/api/b'),
  fetch('/api/c'),
]);

results.forEach(r => {
  if (r.status === 'fulfilled') console.log(r.value);
  else console.error(r.reason);
});
```

Perfect for parallel fetches where some failures are acceptable.
