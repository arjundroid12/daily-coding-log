## 🧠 Challenge of the Day

### Remove Duplicates from Array

**Difficulty:** Easy
**Topic:** Arrays / Sets

**Problem:**

Return a new array with duplicates removed, preserving the order of first occurrence.

**Example:**

```
Input: [1, 2, 2, 3, 4, 4, 5]
Output: [1, 2, 3, 4, 5]
```

**Hint:**

Use a Set to track seen elements, or just convert to Set and back to array.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
// Simple Set approach
function dedupe(arr) {
  return [...new Set(arr)];
}

// Preserve order, filter manually (for objects you can't put in Set)
function dedupeBy(arr, keyFn = (x) => x) {
  const seen = new Set();
  return arr.filter((item) => {
    const key = keyFn(item);
    if (seen.has(key)) return false;
    seen.add(key);
    return true;
  });
}
```

**Explanation:** Sets guarantee uniqueness. The second approach is more flexible — works for objects via a key function.

## 💡 Tip of the Day

**Optional chaining `?.` is your friend**

Replace verbose `&&` chains with `?.`:

```js
// ❌ Verbose
const city = user && user.address && user.address.city;

// ✅ Clean
const city = user?.address?.city;

// Also works for function calls
const result = obj.method?.();  // undefined if method doesn't exist
```
