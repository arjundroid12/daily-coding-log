## 🧠 Challenge of the Day

### Debounce Function

**Difficulty:** Medium
**Topic:** Closures / Async

**Problem:**

Implement a `debounce(func, wait)` function that delays invoking `func` until `wait` ms have elapsed since the last call.

**Example:**

```
const log = debounce(console.log, 300);
log('a'); log('b'); log('c');
// Only 'c' is logged, after 300ms of silence
```

**Hint:**

Use setTimeout. Clear the previous timer on each call so only the last one fires.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function debounce(func, wait) {
  let timer;
  return function (...args) {
    clearTimeout(timer);
    timer = setTimeout(() => func.apply(this, args), wait);
  };
}
```

**Explanation:** Closures over `timer` let us share state across calls. Essential for search-as-you-type, resize handlers, etc.

</details>

## 💡 Tip of the Day

**Use `??` (nullish coalescing) instead of `||` for default values**

`||` treats `0`, `''`, `false`, `NaN` as falsy — usually not what you want for defaults.

`??` only treats `null` and `undefined` as nullish.

```js
const count = response.count || 10;     // ❌ 0 becomes 10
const count = response.count ?? 10;      // ✅ 0 stays 0
```
