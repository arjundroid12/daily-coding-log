## 🧠 Challenge of the Day

### Capitalise Each Word

**Difficulty:** Easy
**Topic:** Strings

**Problem:**

Given a sentence, capitalise the first letter of each word.

**Example:**

```
Input: "the quick brown fox"
Output: "The Quick Brown Fox"
```

**Hint:**

Split by space, map each word to its capitalised version, join back with spaces.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function capitaliseWords(str) {
  return str
    .split(' ')
    .map((word) => word.charAt(0).toUpperCase() + word.slice(1))
    .join(' ');
}
```

**Explanation:** Clean functional approach with split/map/join. The same pattern works for many string transformations.

</details>

## 💡 Tip of the Day

**Avoid `forEach` when you need to `break` or `return`**

`forEach` doesn't support `break` or early return — use `for...of` instead:

```js
// ❌ Can't break out of forEach
arr.forEach(item => {
  if (item === target) return;  // doesn't do what you think!
  console.log(item);
});

// ✅ for...of supports break
for (const item of arr) {
  if (item === target) break;
  console.log(item);
}
```

`.some()` and `.every()` also short-circuit — useful for early-exit searches.
