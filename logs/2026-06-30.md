## 🧠 Challenge of the Day

### Merge Two Sorted Arrays

**Difficulty:** Medium
**Topic:** Arrays / Two Pointers

**Problem:**

Given two sorted arrays `a` and `b`, merge them into a single sorted array.

**Example:**

```
Input: a = [1, 3, 5], b = [2, 4, 6]
Output: [1, 2, 3, 4, 5, 6]
```

**Hint:**

Use two pointers — one per array. Compare current elements, push the smaller one, advance that pointer.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function mergeSorted(a, b) {
  const result = [];
  let i = 0, j = 0;
  while (i < a.length && j < b.length) {
    if (a[i] <= b[j]) result.push(a[i++]);
    else result.push(b[j++]);
  }
  while (i < a.length) result.push(a[i++]);
  while (j < b.length) result.push(b[j++]);
  return result;
}

// O(n + m) time
```

**Explanation:** Exploits the fact that inputs are already sorted. This is the merge step of merge sort.

## 💡 Tip of the Day

**Use `Set` for fast membership checks**

`Set.has()` is O(1), `Array.includes()` is O(n).

```js
// ❌ Slow for large arrays
const allowed = ['admin', 'editor', 'moderator'];
if (allowed.includes(role)) { ... }

// ✅ Fast — O(1) lookup
const allowed = new Set(['admin', 'editor', 'moderator']);
if (allowed.has(role)) { ... }
```

Rule of thumb: if you call `.includes()` more than once on the same array, convert to Set.
