## 🧠 Challenge of the Day

### Reverse a String

**Difficulty:** Easy
**Topic:** Strings

**Problem:**

Reverse a string without using the built-in `.reverse()` method.

**Example:**

```
Input: "hello"
Output: "olleh"
```

**Hint:**

Use two pointers — one at the start, one at the end — and swap characters moving toward the center.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function reverseString(str) {
  const chars = str.split('');
  let left = 0;
  let right = chars.length - 1;
  while (left < right) {
    [chars[left], chars[right]] = [chars[right], chars[left]];
    left++;
    right--;
  }
  return chars.join('');
}

// O(n) time, O(n) space (for the array)
```

**Explanation:** Two-pointer technique — halve the work by going from both ends. Same pattern works for reversing arrays in place.

## 💡 Tip of the Day

**Use `Set` for fast membership checks**

`Set.has()` is O(1), `Array.includes()` is O(n).

```js
// ❌ Slow for large arrays
const allowed = ['admin', 'editor', 'moderator'];
if (allowed.includes(role)) { ... }

// ✅ Fast — O(1) lookup
const allowed = new Set(['admin', 'editor', 'moderator']);
if (allowed.has(role)) { ... }
```

Rule of thumb: if you call `.includes()` more than once on the same array, convert to Set.
