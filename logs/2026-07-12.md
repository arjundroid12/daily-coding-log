## 🧠 Challenge of the Day

### Capitalise Each Word

**Difficulty:** Easy
**Topic:** Strings

**Problem:**

Given a sentence, capitalise the first letter of each word.

**Example:**

```
Input: "the quick brown fox"
Output: "The Quick Brown Fox"
```

**Hint:**

Split by space, map each word to its capitalised version, join back with spaces.

<details>
<summary>🔍 Reveal solution</summary>

```javascript
function capitaliseWords(str) {
  return str
    .split(' ')
    .map((word) => word.charAt(0).toUpperCase() + word.slice(1))
    .join(' ');
}
```

**Explanation:** Clean functional approach with split/map/join. The same pattern works for many string transformations.

</details>

## 💡 Tip of the Day

**Use `Intl` APIs for locale-aware formatting**

Stop hardcoding date/number formats. Use `Intl`:

```js
// Currency
new Intl.NumberFormat('en-IN', { style: 'currency', currency: 'INR' })
  .format(1234567.89);  // '₹12,34,567.89' (Indian grouping)

// Dates
new Intl.DateTimeFormat('en-IN', { dateStyle: 'full' })
  .format(new Date());  // 'Monday, 29 June 2026'

// Relative time
new Intl.RelativeTimeFormat('en', { style: 'short' })
  .format(-1, 'day');  // 'yesterday'
```
